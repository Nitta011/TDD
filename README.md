# TDD
Desenvolvimento orientado a teste
# Trabalho 1 
Cada aluno receberá um conjunto de equações do Geniol e através de uma única classe de *testes* (Junit, Java) deverá criar um teste para resolver as equações.
A classe de teste deverá imprimir as equações e ler as entradas correspondentes as variáveis para resolve-las.
Se a entrada estiver correta o teste passa, se não falha.
Dica: Para tal teste, você não precisa saber o valor das entradas da sua equação e nem da saída!


Dicas da dinâmica do joguinho: https://www.geniol.com.br/logica/equacoes-logicas/
Vídeo: https://www.loom.com/share/e32d0a10e79b475dbe756ac9d720e743

Individual: Vale 2 pontos.
